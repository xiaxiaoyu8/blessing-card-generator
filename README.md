## ✨ 项目亮点

- 🎯 **智能分类** - 按节日（春节/中秋/圣诞等）和对象（长辈/晚辈/朋友）精准匹配
- 🎨 **精美界面** - 基于 Tkinter 的现代简洁 GUI 设计
- 📋 **一键复制** - 选中即复制，发送祝福快人一步
- 🎲 **随机推荐** - 海量祝福语库，每次打开都有新惊喜
- 💾 **离线使用** - 无需联网，本地运行，隐私无忧
- 🚀 **即开即用** - 提供 Windows 可执行文件，无需配置 Python 环境
